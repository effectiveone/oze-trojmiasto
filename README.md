# OZE Trójmiasto

Niezależny portal informacyjny o odnawialnych źródłach energii w Gdańsku, Gdyni i Sopocie.

## Stack
- Astro 4.x (static output)
- Vanilla CSS (newspaper black/white/red theme)
- Cloudflare Pages

## Uruchomienie lokalne
```bash
npm install
npm run dev
```

## Deploy
Cloudflare Pages: build command `npm run build`, publish directory `dist`
